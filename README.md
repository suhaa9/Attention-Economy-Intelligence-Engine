# Attention-Economy-Intelligence-Engine

# Business Problem -

Traditional business metrics such as revenue and market share are lagging indicators. This project investigates whether consumer attention can serve as a leading indicator of future business growth.

# Objective - 
Develop a business intelligence system that:
1. Tracks consumer attention using Google Trends
2. Compares competitive positioning across brands
3. Measures attention momentum
4. Identifies emerging market leaders
5. Forecasts future attention trends
   
# Dataset -
Data Source: Google Trends

Brands Analyzed:

1. OpenAI
2. Canva
3. Grammarly
4. Notion
5. Perplexity AI

# Methodology -

1. Data Collection
2. Extracted search-interest data using PyTrends API
3. Collected weekly attention metrics
4. Data Cleaning
5. Removed partial observations
6. Structured data for analysis
7. Exported clean datasets
8. Exploratory Data Analysis
9. Trend analysis
10. Growth analysis
11. Competitive benchmarking
12. Proprietary Metric
13. Attention Momentum Score

# Formula:

Attention Momentum Score = 0.7 × Average Attention + 0.3 × Growth Rate

This metric evaluates both current market awareness and growth momentum.

# Dashboard Features -

Executive KPIs
Highest Attention Score
Average Attention Score
Highest Growth Rate
Visualizations
Attention Momentum Rankings
Brand Growth Comparison
Brand Attention Trends
Interactive Brand Filter

# Business Insights -

1. OpenAI demonstrated the highest overall market attention.
2. Perplexity AI showed strong momentum growth.
3. Consumer attention can be used as an early signal of market shifts and competitive positioning.
