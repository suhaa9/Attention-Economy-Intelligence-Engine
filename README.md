# Attention-Economy-Intelligence-Engine
Business Problem -
Traditional business metrics such as revenue and market share are lagging indicators. This project investigates whether consumer attention can serve as a leading indicator of future business growth.

Objective - 
Develop a business intelligence system that:

Tracks consumer attention using Google Trends
Compares competitive positioning across brands
Measures attention momentum
Identifies emerging market leaders
Forecasts future attention trends
Dataset

Data Source: Google Trends

Brands Analyzed:

OpenAI
Canva
Grammarly
Notion
Perplexity AI

Methodology -

Data Collection
Extracted search-interest data using PyTrends API
Collected weekly attention metrics
Data Cleaning
Removed partial observations
Structured data for analysis
Exported clean datasets
Exploratory Data Analysis
Trend analysis
Growth analysis
Competitive benchmarking
Proprietary Metric
Attention Momentum Score

Formula:

Attention Momentum Score = 0.7 × Average Attention + 0.3 × Growth Rate

This metric evaluates both current market awareness and growth momentum.

Dashboard Features -

Executive KPIs
Highest Attention Score
Average Attention Score
Highest Growth Rate
Visualizations
Attention Momentum Rankings
Brand Growth Comparison
Brand Attention Trends
Interactive Brand Filter

Business Insights -

OpenAI demonstrated the highest overall market attention.
Perplexity AI showed strong momentum growth.
Consumer attention can be used as an early signal of market shifts and competitive positioning.
